The "User Portfolio" project, developed during my internship, is a web application that allows users to register and log in to create their own portfolios. Work 03

Travalista

"Travlista" is a web application developed during my internship, which emphasizes the use of REST APIs to facilitate interaction between users and the platform. The application allows tourists to upload details about the places they have visited, including photos, descriptions, and locations. Additionally, users can view and explore places shared by others, creating a community-driven travel experience. The backend, built with Python and Django, utilizes REST APIs to handle data exchange and integration, while SQLite3 manages the database. The front end, designed with HTML, CSS, JavaScript, and Bootstrap, ensures a seamless and interactive user experience.

Work 04

BookHub

"Bookhub" is a web application developed during my internship, prominently featuring React for a dynamic and interactive front end. The platform supports both librarians and readers, allowing them to log in and access different functionalities. Readers can view detailed information about books, including their locations within the library, while librarians have the ability to manage book details and update their locations. The backend is powered by Python and Django, with SQLite3 managing the database. The integration of React enhances the user interface, making it responsive and engaging, complemented by HTML, CSS, JavaScript, and Bootstrap for overall design and functionality.

Work 05

E-Hospitality

"E-Hospitality" is a personal project that provides a comprehensive platform for managing hospital appointments and patient information. Both patients and doctors can register and log in to the system. Doctors can view their scheduled appointments, including details about which patients have booked to see them. Patients, on the other hand, can view available doctors, their schedules, and book appointments, providing details about their health concerns and personal information. The application is developed using Python and Django for the backend, with SQLite3 for database management. The front end is designed with HTML, CSS, JavaScript, and Bootstrap, offering a user-friendly interface for both patients and doctors.
